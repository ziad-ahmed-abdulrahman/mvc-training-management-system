# Training Management System

A simple ASP.NET Core MVC project for managing training courses, departments, instructors, and trainees.

## Features

- User authentication (Login / Register / Logout)
- Manage Departments
- Manage Courses
- Manage Instructors
- Manage Trainees
- Confirmation dialogs for sensitive actions (delete, logout)
